Markdown

Markdown是什么？

Markdown 是一种轻量级标记语言，创始人为John Gruber和Aaron Swartz。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。

摘自维基百科Markdown条。

Markdown有啥好处？

正如上文所述，Markdown具有：

  1. 易学易用；

  2. 方便转换为HTML等其他格式；

  3. 支持广泛，便于维护；

等等特性。
